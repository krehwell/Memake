# Memake
Yeet, yeet, yeet!.

```c++
#include "Memake.h"
Memake mmk(800, 800, "memake window");

void draw() {
    mmk.drawRect(300, 300, 100, 100, Colmake.red); // draw a rectangle with red color
}

int main() {
    mmk.update(draw);
    return 0;
}
```

## Dependencies
- [SDL2](https://www.libsdl.org/)
